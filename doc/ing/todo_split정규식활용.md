# split 정규식 활용

## 목표
`String.split()` 및 정규식을 활용한 자주 쓰이는 패턴 예제 작성

## 예제 항목

### split 기본 패턴
- 공백 구분 (단순 공백)
- 연속 공백(탭 포함) 구분 — `\s+`
- 숫자 추출 — `\D+`로 분리
- 영문자 추출 — `\d+`로 분리
- 단일 구분기호 분리 (쉼표)
- 복합 구분기호 분리 — `[,;|/]`

### 실전 활용 패턴
- 숫자만 포함 여부 확인 — `^\d+$`
- 이메일 형식 검증
- 한국 휴대폰 번호 검증 — `^01(?:0|1|[6-9])-...`
- 비밀번호 규칙 검증 — 전방 탐색(`(?=...)`) 활용
- 연속 공백 정규화 — `\s+` + `replaceAll`

## 정규식 핵심 개념 정리

### 기본 문자 클래스
- `.` — 임의의 한 문자 (줄바꿈 제외)
- `\d` — 숫자 (`[0-9]`)
- `\D` — 숫자가 아닌 것
- `\w` — 알파벳 대소문자, 숫자, 언더바
- `\W` — `\w`가 아닌 것
- `\s` — 공백 문자 (스페이스, 탭, 줄바꿈)
- `\S` — 공백이 아닌 문자

### 수량자
- `*` — 0번 이상
- `+` — 1번 이상
- `?` — 0 또는 1번
- `{n}` — 정확히 n번
- `{n,}` — n번 이상
- `{n,m}` — n번 이상 m번 이하

### 경계 및 위치
- `^` — 문자열 시작
- `$` — 문자열 끝
- `\b` — 단어 경계
- `|` — OR

### 자바 주의사항
- 역슬래시를 두 번 써야 함: 정규식 `\d` → 자바 문자열 `"\\d"`
