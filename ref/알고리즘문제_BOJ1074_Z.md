# BOJ 1074 — Z

- 링크: https://www.acmicpc.net/problem/1074
- 시간 제한: 0.5초 / 메모리: 512MB
- 난이도: 실버 I
- 분류: 재귀, 분할정복

## 문제 설명

2^N × 2^N 크기의 2차원 배열을 Z 모양으로 순서대로 방문한다.
2×2는 왼쪽 위 → 오른쪽 위 → 왼쪽 아래 → 오른쪽 아래 순서.
N > 1이면 배열을 4등분 후 각 구역을 재귀적으로 같은 방식으로 방문한다.

r행 c열을 몇 번째로 방문하는지 출력하라.

## 입력

N r c (1 ≤ N ≤ 15, 0 ≤ r, c < 2^N)

```
2 3 1
```

## 출력

```
11
```

## 구현 아이디어

- 재귀: 4개 구역 중 목표 칸이 속한 구역만 재귀, 나머지는 (2^(N-1))^2 만큼 건너뜀
- 수학: 비트 연산으로 방문 순서를 직접 계산
