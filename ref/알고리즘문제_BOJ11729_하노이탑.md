# BOJ 11729 — 하노이 탑 이동 순서

- 링크: https://www.acmicpc.net/problem/11729
- 시간 제한: 1초 / 메모리: 256MB
- 난이도: 실버 III
- 분류: 재귀

## 문제 설명

3개의 기둥과 N개의 원판이 있다. 원판은 크기가 모두 다르고, 처음에는 1번 기둥에 크기 순서대로 쌓여 있다.
아래 규칙을 지키면서 1번 기둥의 원판을 모두 3번 기둥으로 옮겨라.
- 한 번에 한 개의 원판만 이동 가능
- 큰 원판이 작은 원판 위에 올 수 없음

최소 이동 횟수와 이동 순서를 출력하라.

## 입력

N (1 ≤ N ≤ 20)

```
3
```

## 출력

첫째 줄: 총 이동 횟수 K
다음 K줄: 이동마다 `A B` (A번 기둥 → B번 기둥)

```
7
1 3
1 2
3 2
1 3
2 1
2 3
1 3
```

## 구현 아이디어

- 재귀: f(n, from, to, via) → f(n-1, from, via, to) + move(from→to) + f(n-1, via, to, from)
- 이동 횟수: 2^N - 1
