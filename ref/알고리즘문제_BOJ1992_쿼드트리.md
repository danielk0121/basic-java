# BOJ 1992 — 쿼드트리

- 링크: https://www.acmicpc.net/problem/1992
- 시간 제한: 2초 / 메모리: 128MB
- 난이도: 실버 I
- 분류: 재귀, 분할정복

## 문제 설명

N×N 흑백 영상을 쿼드트리로 압축한다.
- 전체가 0이면 `0` 출력
- 전체가 1이면 `1` 출력
- 0과 1이 섞여 있으면 4등분 후 재귀 압축, 결과를 `(...)` 로 감쌈
- 분할 순서: 왼쪽 위 → 오른쪽 위 → 왼쪽 아래 → 오른쪽 아래

## 입력

N (1 ≤ N ≤ 64, 2의 거듭제곱), 이후 N×N 행렬

```
8
11110000
11110000
00011100
00011100
11110000
11110000
11110011
11110011
```

## 출력

```
((110(0101))(0010)1(0001))
```

## 구현 아이디어

- 재귀: 구역 전체가 같은 값인지 확인 후 분기
- StringBuilder에 누적하며 재귀 출력
