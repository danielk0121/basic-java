# BOJ 2448 — 별 찍기 - 11

- 링크: https://www.acmicpc.net/problem/2448
- 시간 제한: 1초 / 메모리: 256MB
- 난이도: 골드 V
- 분류: 재귀, 분할정복

## 문제 설명

삼각형 별 패턴을 재귀적으로 출력한다. N은 3×2^k (3, 6, 12, 24, ...).

크기 3 기본 패턴:
```
  *
 ***
*****
```
크기 6은 기본 삼각형 3개를 아래처럼 배치한다:
```
     *
    ***
   *****
  *     *
 ***   ***
***** *****
```
크기 12, 24도 같은 방식으로 재귀 확장된다.

## 입력

N (3×2^k, 0 ≤ k ≤ 10)

```
24
```

## 출력

N줄의 삼각형 별 패턴

## 구현 아이디어

- 재귀: 2D char 배열에 삼각형을 채운 뒤 출력
- 재귀: 위 삼각형 + 왼쪽 아래 삼각형 + 오른쪽 아래 삼각형으로 분할
